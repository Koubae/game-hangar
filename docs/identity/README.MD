Auth
----


There are 3 Providers categories


* Managed
* anonymous
* Platform
* Managed


The **managed** + **anonymous** auth providers are all auth methods own
by the company and is the company responsible to handle 
end-to-end authorization.

All the other provider category are pretty much 3rd parties.
